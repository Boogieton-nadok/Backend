# 📚 나독 (Nadok)

> 감정 기반 도서 캐릭터 매칭 · 개인 서재 관리 · AI 도서 토론 챗봇을 하나로 묶은 독서 경험 앱

---

## 주요 기능

- **감정 기반 캐릭터 매칭** — 현재 감정 상태를 입력하면 어울리는 도서 속 캐릭터를 추천
- **나만의 서재** — 읽은 책, 읽고 싶은 책을 저장하고 개인 서재를 관리
- **AI 도서 토론 챗봇** — 책에 관한 내용을 AI와 자유롭게 토론하고 다양한 시각을 나눔

---

## 기술 스택

| 분류 | 기술 |
|------|------|
| Language | Java |
| Framework | Spring Boot |
| AI | Gemini API |
| Database | MySQL |

---

## 커밋 컨벤션

| 타입 | 설명 |
|------|------|
| `feat` | 새로운 기능 추가 또는 기존 기능 개선 |
| `fix` | 버그 수정 |
| `refactor` | 코드 리팩토링 (기능 변화 없이 구조 개선) |
| `doc` | 문서 작업 (README 등) |
| `test` | 테스트 코드 추가 또는 수정 |
| `build` | 빌드 관련 파일 수정 (예: yml, properties 등) |
| `perform` | 성능 개선 |
| `clean` | 불필요한 코드 제거, 정리 |
| `design` | UI/UX 디자인 작업 또는 개선 |
| `style` | 코드 스타일 변경 (세미콜론, 들여쓰기 등) – 기능 변화 없음 |
